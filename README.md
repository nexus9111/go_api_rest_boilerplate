# REST API in Go

## Description

This is a boilerplate for a REST API using Go.

## Installation

```console
$ cp .env.example .env
/* edit .env as needed */
$ go run app.go 
```

You should see the following output:

```console
$ go run app.go
Server started on port 3000
```

## Usage

make sure app is running
```console
$ curl -X GET http://localhost:3000/
```
